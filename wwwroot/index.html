Sistema de gestión de farmacia

Introduccion:

Este portal cuenta con diferentes secciones donde tendra un resumen de las facturas vendidas en un Dashboard, una sección para realizar una transacción de tipo venta, otra para dar de alta productos y finalmente una sección para modificar o eliminar productos si lo desea.

El uso de las distintas secciones está restringido según su rol asignado (usuario registrado), será capaz de acceder a:

Datos tecnicos:

El front-end se desarrolló con HTML5, CSS3 y Bootstrap para la maquetación. La lógica en JavaScript consume la API del back-end mediante peticiones HTTP (GET, POST, PUT, DELETE), actualizando dinámicamente la interfaz según las respuestas del servidor.

En el back-end se implementó una API RESTful en C# sobre ASP.NET, utilizando SQL Server como base de datos. La autenticación se realiza mediante JWT, y a partir del rol incluido en el token (administrador o vendedor) se controla el acceso a los endpoints y a las distintas secciones del sistema.
