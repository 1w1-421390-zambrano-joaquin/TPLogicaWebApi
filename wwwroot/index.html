Sistema de gestión de farmacia
Introduccion:

Este portal cuenta con diferentes secciones. Un Dashboard donde podrá visualizar un resumen de las facturas emitidas, una sección ventas para realizar la transacción de la misma y secciones para realizar alta, baja y modificacion de productos en el sistema.

El uso de las distintas secciones está restringido según su rol asignado (usuario registrado), será capaz de acceder a:

Datos tecnicos:

El front-end se desarrolló con HTML5, CSS3 y Bootstrap para la maquetación. La lógica en JavaScript consume la API del back-end mediante peticiones HTTP, actualizando dinámicamente la interfaz según las respuestas del servidor.

En el back-end se implementó una API RESTful en C# sobre ASP.NET, utilizando SQL Server como base de datos. La autenticación se realiza mediante JWT, y a partir del rol incluido en el token (administrador o vendedor) se controla el acceso a los endpoints y a las distintas secciones del sistema.
